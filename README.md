A Snake game implementation in C++.

## Gameplay

- Snake grows with each food eaten (+10 points)
- Speed increases every 50 points (max 10 levels)
- Collision detection with walls and self

## Controls

- **R** - Start game
- **Q** - Quit game
- **WASD** - Move snake

## Requirements

- C++20 compiler
- CMake 3.20+
- Python 3.x
